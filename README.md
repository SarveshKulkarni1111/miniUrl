# miniUrl

A **URL shortening service** built with **Node.js, Express, MySQL, and React**.

This project allows users to create short URLs for long links, track the number of redirects, and manage URLs with a simple and minimal interface.

---

## Features

- Shorten long URLs
- View all shortened URLs in a table
- Copy short URL to clipboard
- Delete URLs
- Track redirect counts
- Minimal React frontend with Tailwind CSS
- Backend API built with Express and MySQL

---

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MySQL
- **HTTP Client:** Axios
